Signa é um projeto de desenvolvimento multiplataforma, com o intuito de ser uma ferramenta capaz de diminuir a barreira linguística entre os estudantes que possuem problemas de comunicação por causa de surdez, com profesores, de maneira rápida, fácil e tecnológica.
